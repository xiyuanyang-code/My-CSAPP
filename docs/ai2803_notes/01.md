# 计算机的基本结构

## 冯诺依曼结构

> 计算机领域最精妙的设计是**抽象**

一种重要的设计思想：**程序和数据**同样通过二进制的方式被存放在存储器。
- 程序像数据一样被存储，而不是通过手动的**开关连线控制**
    - 如何实现？一套非常具有前瞻性的蓝图
        - IO 设备、存储器、控制器、运算器
        - 数据线路主要是数据的流动方向
        - **控制线路**提供了抽象，保证不同的计算任务可以通过不同的程序被统一的表示
    - 运算器、控制器、存储器、输入输出设备


### 南北桥架构

CPU 的运算速度远高于存储等电脑上的其他设备，因此在早期的计算机结构设计中，使用**南北桥**的架构设计来分别处理不同速度的数据流，例如北桥处理速度较快的数据流而南桥处理速度较慢的数据流。

但是随着半导体工艺的飞速进展，南北桥结构已经演变成更低时延的结构：

- 南桥被 CPU 吞并，内存和 CPU 之间存在专门的通道
- 北桥被整合成 PCH，并集成了链接外部设备的多个接口

## 计算机读取指令并执行的具体步骤

- 取指：
    - 控制器将指令地址送往存储器
    - 存储器按照地址读出内容，送回控制器
    - 控制器更新下一步指令地址
- 译码：
    - 控制器解译指令的操作
    - 控制器向设计的部件发出控制信号
- 执行：
    - 控制器从通用寄存器或者存储器中取出操作数
    - 控制器控制运算器进行指令执行
- 回写：
    - 将运算结果写入寄存器或者存储器
